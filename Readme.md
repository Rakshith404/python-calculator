# Python Calculator

A simple calculator built using Python.

This project includes:
- A **CLI Calculator** (runs in terminal)
- A **GUI Calculator** (built using Tkinter)

---

## GUI Preview

![Calculator GUI](screenshot.png)


## Features

### CLI Calculator
- Basic arithmetic operations (+, -, *, /)
- Supports functions like:
  - sqrt, power, sin, cos, tan, log
- Calculation history
- Uses previous answer with `ans`
- Safe evaluation (restricted functions)

### GUI Calculator
- Clean calculator interface
- Button press animation
- Keyboard input support
- Backspace (delete one digit)
- Display limit like real calculators
- Scientific notation for large results
